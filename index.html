agel-table | 使用文档

基于 element-ui table 的二次封装加强组件

起步 →

极简的思想

面向对象, 参数集中处理, 以一个 table 对象来完成所有的操作

高度的灵活

支持 element-ui table 组件的所有 api, slot, event，且不用担心版本升级问题

快捷的使用

解决了 element-ui table 的潜在 bug, 集成了更方便好用的扩展功能
