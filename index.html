agel-table | 使用文档

基于 element-ui table 的二次封装组件

起步 →

极简的思想

面向对象, 参数集中处理, 以一个 table 对象来完成所有的操作

高度的灵活

支持 element-ui table 组件的所有 api, slot, event

快捷的使用

解决了 element-ui table 的潜在 bug, 集成了更方便好用的扩展

# 如此简单

<template>
  <agel-table v-model="table"> </agel-table>
</template>
<script>
  export default {
    data() {
      return {
        table: {}
      };
    }
  };
</script>
